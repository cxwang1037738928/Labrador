## To run locally, do the following:

## Setup & Run

### 1. Clone the repository
```bash
git clone https://github.com/cxwang1037738928/Labrador.git
cd project_root
```

### 2. Start the backend
```bash
cd backend
npm run dev
```

### 3. Start the mobile app (in a new terminal)
```bash
cd mobile_app
npx expo start
```
